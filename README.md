# ytdlp-ai-summarizer
This bash script - android application downloads subtitles for playlists and videos via yt-dlp then uses your local Ollama setup to summarize it all. With the android app you can also ask follow up questions and add multiple videos in a single instance.
Note: this content is AI assisted. It's fine as long as it works.

To use this in android, you'll need Termux with external applications enabled, access to all files and yt-dlp installed. You might also want to install ollama and pull the LLM that fits your system, or you can connect it to a local instance (e.g. your computer)
